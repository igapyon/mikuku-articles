# 002 きっかけ

## 主題

GPT-5.5 では、粗い Agent Skills でも「場」として機能していた。

## キーワード

- 中ぐらいの品質
- 量で動く
- 粗い Markdown
- 古いメモ
- 文脈の場
- 意図補完
- Qwen3 併用
- 頑張ることと安定すること

## 図解ポイント

```text
粗い資料
  + 古いメモ
  + 粒度違い
        ↓
GPT-5.5 が意図を補う
        ↓
実務的に進む
```

## 対比

| 期待 | 現実 |
| --- | --- |
| GPT-5.5 だけで十分 | コストや運用制約がある |
| Qwen3 も同じように動く | 頑張るが、安定には工夫が必要 |

## 吹き出し案

「場として読めるモデルと、安定運用は別に考えたいです…」
