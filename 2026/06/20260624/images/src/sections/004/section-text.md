# 004 Agent Skills には、コンテンツ型と駆動型がある

## 主題

Agent Skills を「コンテンツ型」と「駆動型」に分けて考える。

## コンテンツ型

- 背景
- 価値観
- 判断基準
- 用語
- 文体
- 事例
- 設計思想
- 過去の経緯

役割: **作業空間に文脈を置く**

## 駆動型

- 読む順番
- 確認タイミング
- 作業の刻み方
- 進め方

役割: **作業手順を補う**

## 図解ポイント

```text
共通に保つ:
  コンテンツ型 Skill

Qwen 用に fork してよい:
  駆動型 Skill
```
