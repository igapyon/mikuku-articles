# セクション別グラレコ整理テキスト

## 見出し

4. 生成AIが変える世界に、計画を固定せず対応する

## 対応する価値

- Responding to change over following a plan
- 計画に従うだけより、変化への対応

## 主題

- 生成AIで変更コストの一部は下がる
- 変化は要求変更だけでなく、社会・業務・期待にも広がる
- 変化を小さく分けて試すことが重要

## 2つの変化

```text
1. 生成AIを使って、変更や試作にすばやく対応
2. 生成AIによって変わる社会・業務・顧客期待に対応
```

## 注意点

- 変化が大きすぎるとコストは跳ねる
- 速く作れるほど、切り方と順番が重要
- 変える判断と変えない判断の両方が大事

## 視覚キーワード

- 小さく分ける
- 試す
- 計画を固定しない
- 変化の波
- 専門家が効く切り分け
