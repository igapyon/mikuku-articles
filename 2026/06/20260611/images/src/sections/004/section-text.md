# セクション別グラレコ整理テキスト

## 見出し

2. 後で動くものから、すぐに、あるいは目の前で動くソフトウェアを実現

## 対応する価値

- Working software over comprehensive documentation
- 分厚い文書より、動くソフトウェア

## 主題

- 動くソフトウェアが「後で見るもの」から「今一緒に触るもの」へ変わる
- 速さだけでなく、違和感や期待を見つけやすくなる

## Before / After

```text
Before:
打ち合わせ → 持ち帰り → 実装 → 次回レビュー

After:
対話 → その場で試作 → 触る → 気づく → 直す
```

## 図解キーワード

- 目の前の試作品
- 画面のたたき台
- 触りながら確認
- 言葉にならない違和感
- 未来の小さな試作品

## 視覚構成

- 左: 時間差のあるレビュー
- 右: その場で動くプロトタイプ
- 中央: 「動くものが対話の場にいる」
