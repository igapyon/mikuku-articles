# セクション別グラレコ制作用整理テキスト

## 主題

- おわりに
- アジャイルを説明するときは、スクラムやツールの前に原典と歴史へ戻る

## 図解キーワード

- まず宣言へ戻る
- 4つの価値を読む
- スクラム / XP / カンバン / ツールを見る
- 価値観の具体化
- 入口が見えやすくなる

## 流れ

```text
スクラムのイベント
チケット管理
ツール運用
  ↓ いきなり入ると狭い
原典と歴史に戻る
  ↓
何に違和感を持ったのか
何を大事にしたのか
  ↓
実践やツールの意味が見える
```

## 対比

| 前に出やすいもの | 土台として見るもの |
|---|---|
| スクラムイベント | アジャイル宣言 |
| チケット管理 | 4つの価値 |
| ツール運用 | 歴史と問題意識 |

## 最終メッセージ

- まず宣言に戻る
- 4つの価値を読む
- そのうえで実践を見る
