# セクション別グラレコ制作用整理テキスト

## 主題

- 12の原則も宣言の一部として読む
- 4つの価値を、日々の開発でどう扱うか

## 図解キーワード

- 4つの価値
- 12原則
- 入口の扉
- その先の廊下
- 日々の開発

## 12原則の短いラベル

- 早く継続的に価値を届ける
- 変化する要求を受け入れる
- 短い間隔で届ける
- 日々協力する
- 意欲ある人を信頼する
- 直接対話を重視する
- 動くものを進捗の尺度にする
- 持続可能なペース
- 技術的卓越性とよい設計
- シンプルさ
- 自己組織的なチーム
- 定期的なふりかえりと調整

## 構造

```text
4つの価値
  ↓ 判断の軸
12原則
  ↓ 日々の行動へ
開発の進め方
```

## 注意

- ここでの対応関係は公式対応表ではない
- 原典に戻って確かめる姿勢が大事
