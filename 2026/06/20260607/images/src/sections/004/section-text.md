# セクション004 グラレコ制作用整理テキスト

## 主題

- 「動くものが作れた」の次を見る
- 実行するたびの消費を評価する
- 短くするだけではなく、結果と消費の比を見る

## 中心式

```text
実行時トークン効率
  =
実現できる実行結果
  /
実行時の消費トークン
```

## 重い状態

- 巨大な前提説明
- 長いログ
- 毎回同じ自然文手順
- 長すぎる説明出力

## 軽い状態

- Agent Skills
- テンプレート
- CLI 抽出
- MCP
- cache-friendly prefix
- 必要粒度の出力

## 伝えたい一文

- 必要な文脈は保ち、無駄な入力・出力・再説明・推論を減らす。
