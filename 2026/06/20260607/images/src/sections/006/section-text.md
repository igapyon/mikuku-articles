# セクション006 グラレコ制作用整理テキスト

## 主題

- プロンプトだけで頑張らない
- 役割を分けて、AIに渡す文脈を小さくする

## 役割分担

| 要素 | 役割 |
| --- | --- |
| プロンプト | 目的・判断基準・作業境界 |
| Agent Skills | 何度も使う知識・手順・文体 |
| CLI | 検索・抽出・ビルド・テスト・集計 |
| MCP | 外部サービス・共有データへの定型アクセス |

## 構造

```text
AI
  判断 / 設計 / 編集 / つなぎ

ツール
  検索 / 抽出 / 検証 / 定型操作
```

## 伝えたい一文

- たくさん読める時代だからこそ、必要なところだけ渡す設計が大事。
