# セクション010 グラレコ制作用整理テキスト

## 主題

- 省トークンは単なる節約ではない
- AIに無理をさせない整え方
- 軽い実行時トークンで正しく動く環境を作る

## まとめ構造

```text
入力を整理
出力を絞る
Skills に知識を置く
CLI に検索・検証を任せる
MCP に定型アクセスを任せる
固定前提を caching へ寄せる
文脈負債を減らす
```

## 最終メッセージ

- 短いプロンプトそのものが偉いわけではない
- 必要なものを残す
- 毎回いらないものを外す
- 控えめな設計が強い

## 伝えたい一文

- 軽く、正しく、安定して動く生成AIワークフローへ。
