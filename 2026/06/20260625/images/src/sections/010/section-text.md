# 010 まだ課題は残っている

## 主題

- Skill 化しても状態管理は自動で解決しない
- ファイルは古くなる
- 書きすぎても足りなくても迷う

## 課題

| ファイル | 古くなる/書きすぎると |
| --- | --- |
| `GOAL.md` | 古い目的に引っ張られる |
| `TODO.md` | 現在地がずれる |
| `DECISIONS.md` | 長い作業ログになる |
| `HANDOFF.md` | 次の人やAIが過去状態を現在だと思う |

## 中心メッセージ

- ファイルを置くだけでは足りない
- 軽く、でも更新されること
- 必要な判断が残っていること
- AIが読みやすく、人間も確認できること

## グラレコ構図

- 4ファイルに「古い」「長すぎる」「未更新」の警告ラベル
- 右下に「軽く、でも更新」のバランス秤
- みくく: 「置くだけではなく、育てる必要があります…」
