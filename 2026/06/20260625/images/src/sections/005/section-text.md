# 005 Setup と Resume を分けた

## 主題

- Setup と Resume を明確に分ける
- Setup は作る・整える
- Resume はまず読む・現在地を整理する

## 対比

| Setup | Resume |
| --- | --- |
| 状態管理ファイルを作る | 既存状態を読む |
| 既存ファイルに合わせて整える | 目的、次タスク、詰まりを整理 |
| 新規リポジトリ向け | 作業再開向け |

## Resume で読むもの

- `git status --short`
- `README.md`
- `TODO.md`
- `GOAL.md`
- `DECISIONS.md`
- `HANDOFF.md`

## 中心メッセージ

```text
再開時には「作る」より先に「読む」
```

## グラレコ構図

- 左: Setup の工具箱
- 右: Resume の虫眼鏡
- 中央: 「まず読む」の大きな矢印
- みくく: 「いきなり片づけず、まず見てほしいのです…」
