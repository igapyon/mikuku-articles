# 006 既存の TODO.md を壊さない

## 主題

- 既存 `TODO.md` を AI 都合で置き換えない
- 人間用メモやリリース作業を尊重する
- 必要なら専用セクションだけ足す

## 既存 TODO の用途

- 人間用の作業メモ
- リリース前の残作業
- 記事管理
- 画像回収の進捗
- 将来やりたい改善

## 追加する場所

```markdown
## AI Agent Current Tasks
```

## 図解

```text
既存 TODO.md
  ├─ 人間用メモ
  ├─ リリース作業
  └─ ## AI Agent Current Tasks
       ├─ current task
       ├─ blocker
       └─ retry log
```

## 中心メッセージ

- 人間が使っている棚を勝手に並べ替えない
- AI用の小さな引き出しだけ足す

## グラレコ構図

- 棚アイコンの一部に小さな引き出しを追加
- `TODO.md` を丸ごと上書きする矢印にはバツ印
- みくく: 「壊さず、足すだけにします…」
