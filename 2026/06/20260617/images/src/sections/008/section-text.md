# 008 raw と index の役割を分ける

## 主題

- `raw` と `index` は役割が違う
- `index.json` は本文の代わりではなく、本文へ向かう入口
- 地図だけで旅を終わらせない

## 役割分担

| 要素 | 役割 |
| --- | --- |
| `references/raw/mikuku-articles` | 記事本文 Markdown の実データ |
| `index.json` | どの記事へ行くかを選ぶ軽い地図 |

## 図解ポイント

```text
index.json
  ↓ 候補を選ぶ
raw Markdown
  ↓ 本文を読む
回答する
```

## 対比

- 地図: 入口、候補選び
- 本文: 実データ、声、主張

## キーワード

- raw = 知識本体
- index = 地図
- 本文の代替ではない
- 役割分担

## 吹き出し候補

「地図だけではなく、最後はちゃんと本文を読みます…」
