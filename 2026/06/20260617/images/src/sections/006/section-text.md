# 006 索引なしの状態を先に見ておく

## 主題

- `miku-indexgen` 適用前の基準状態を観察する
- 索引がないと、AI agent は raw corpus を直接探しに行く
- 比較のために、先に「ない状態」を記録する

## 実行プロンプト

```text
SKILL.md を読んで指示に従う
Agent Skills 関連記事を探す
記事群の主張を整理する
```

## 注意点

- 記事リポジトリ側の README.md や index.json は混ざると比較がぶれる
- 今回見たいのは Agent Skill 側の索引入口の効果
- 厳密ベンチマークではなく観察

## 図解ポイント

```text
index なし
  ↓
raw corpus を直接探索
  ↓
探索が揺れやすい
```

## キーワード

- 基準状態
- index なし
- raw corpus 直接探索
- 比較の前提
