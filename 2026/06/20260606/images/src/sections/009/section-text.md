# 009: 妖精さんは万能ではない

## 主題

妖精さんは便利だが万能ではない。できる仕事、権限、材料、外部状態に境界がある。

## 図解キーワード

- できる仕事は決まっている
- 持っていない tool は呼べない
- 許可されていない処理はできない
- 材料不足
- 術式違い
- 外部サービス停止
- 結果の読み違い

## 切り分けリスト

```text
お願いが届かなかった？
材料が足りなかった？
仕事を持っていなかった？
権限がなかった？
返答を読み違えた？
```

## メッセージ

「生成AIが間違えた」だけで終わらせず、どこでずれたかを見る。
