# 004 速読しやすい入口

## 主題

入口は、AI agent がどこを深く読むべきか決めるための目印。

## まず見るもの

- SKILL.md
- README.md
- TODO.md
- ファイル名
- ディレクトリ名
- front matter
- 見出し
- テスト名
- 関連記事リンク

## 探すもの

- キーワード
- 関数名
- タグ
- 発火条件
- 制約

## 深く読むもの

- 変更に必要だと判断した周辺

## 図解の対比

```text
入口がある
  → 見つけられる
  → 戻ってこられる

入口がない
  → 作業文脈に入りにくい
  → 存在しないものに近くなる
```

## 比喩

- 小さな付箋
- 「ここから読めば大丈夫」という目印

## 吹き出し案

「入口は飾りではなく、戻ってくるための目印なのです…」
