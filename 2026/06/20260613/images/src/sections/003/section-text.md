# 003 AI agent は速読する

## 主題

AI agent は、最初から全部を精読せず、まず速読して読む場所を決める。

## まず見るもの

- SKILL.md
- README.md
- TODO.md
- ファイル名
- front matter
- 見出し
- rg 検索

## 深く読むもの

- 発火した Agent Skills の SKILL.md
- 変更対象の仕様
- 安全上の制約
- テストやビルドの確認方法

## 図解の流れ

```text
全部を読む
ではなく
  ↓
入口を見る
  ↓
対象と方針を決める
  ↓
必要な場所を深く読む
```

## 制約

- 時間
- トークン
- コンテキスト窓
- 大事な文脈が押し出されるリスク

## 吹き出し案

「まず地図を見てから、必要な棚へ向かう感じなのです…」
