# 001 はじめに

## 主題

AI agent の読み方を「速読」として捉える入口。

## キーワード

- AI agent
- コード
- 生成AIプロンプト
- 速読
- 入口
- 見出し
- 必要な場所へ移動
- 大事な場所へ戻る

## 図解の流れ

```text
リポジトリ / 記事
  ↓
入口を探す
  ↓
見出しを見る
  ↓
必要そうな場所へ移動
  ↓
あとで大事な場所へ戻る
```

## 対比

| 誤解 | 本文の意味 |
| --- | --- |
| 雑に読む | 全体の形を先につかむ |
| 全部を同じ深さで読む | 必要な場所へ段階的に近づく |

## 吹き出し案

「あ、あの…速読は雑読みではなく、戻るための読み方なのです…」
