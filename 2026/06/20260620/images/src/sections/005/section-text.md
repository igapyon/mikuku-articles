# Section Graphic Recording Text

## 見出し

蒸留 Markdown を作るプロンプト

## 主題

「記事を要約して」ではなく、資料群の中心概念、主張、観測、未確定点、raw に戻る条件を整理するプロンプトを作る。

## キーワード

- 要約しない
- 回答テンプレートにしない
- context artifact
- 中心概念
- 主要な主張
- 観測されたこと
- まだ断定できないこと
- 元記事へ戻るべき場合
- 注意点

## 図解

```text
raw 記事群
  ↓
中心概念 / 主張 / 観測 / 未確定点
  ↓
agent-skills-context-map.md
  ↓
人間と AI agent が共有
```

## 作成ルール

- 元記事にない主張を足さない
- 断定しすぎない
- 記事ごとの要約にしない
- 手順だけに寄せない
- raw に戻る手がかりを残す

## 吹き出し案

「要約ではなく、意味の地図として整理してもらうのです…」
