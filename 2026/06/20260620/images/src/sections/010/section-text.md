# Section Graphic Recording Text

## 見出し

おわりに

## 主題

`index.json`、蒸留 Markdown、raw 記事本文の役割を分けると、AI agent と人間が同じ全体像を見てから歩き出せる。

## キーワード

- `index.json`
- ファイルの地図
- 蒸留 Markdown
- 意味の地図
- raw 記事本文
- 一次資料
- 役割分担
- raw に戻る前の意図
- 同じ景色を見る

## 図解

```text
index.json:
  どのファイルがあるかを見つける

references/distilled/:
  資料群の意味構造を先につかむ

references/raw/:
  主張や根拠を確認するときに戻る
```

## 結論

- raw を読まなくてよくすることが目的ではない
- どの資料を、どんな意図で読むかをはっきりさせる
- 資料群の見方がそろう
- raw へ戻る判断が明示される

## 吹き出し案

「同じ景色を見てから歩き出せると、少し迷いにくいのです…」
