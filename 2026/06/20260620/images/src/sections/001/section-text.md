# Section Graphic Recording Text

## 見出し

はじめに

## 主題

蒸留 Markdown は raw 本文の代替ではなく、記事群の内容構造を先に見渡すための「意味の地図」。

## キーワード

- step2: `index.json`
- step3: 蒸留 Markdown
- raw の代替ではない
- 意味の地図
- どの raw を読むか
- 何のために読むか
- 根拠を薄くしない

## 図解

```text
index.json
  = ファイルの地図

蒸留 Markdown
  = 意味の地図

raw Markdown
  = 根拠を確認する本文
```

## 対比

| 違うもの | 作りたいもの |
| --- | --- |
| 本文を読まなくてよい近道 | raw に戻る前の見通し |
| 便利な要約 | 内容構造の地図 |
| トークン削減だけ | 読む目的を絞る |
