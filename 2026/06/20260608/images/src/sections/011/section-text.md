# GitHub Copilot は、開発者向けなので別枠で見る

## 主題

GitHub Copilot は、Microsoft 365 側ではなく開発者向けのコード支援として別枠で見る。

## 使う場所

- GitHub
- VS Code
- JetBrains IDE
- 開発環境

## 主な用途

- コード補完
- コード生成
- テスト作成
- コード説明
- リファクタリング支援
- 開発中の質問

## 注意点

- Office作業ではない
- 社内文書整理ではない
- 利用量や課金の見え方に注意

## 吹き出し

「同じ Copilot でも、コードを書く部屋は別なのです…」
