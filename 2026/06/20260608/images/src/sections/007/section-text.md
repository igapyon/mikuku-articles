# Copilot Chat は、利用者視点では RAG のように見える

## 主題

Microsoft 365 Copilot ライセンスありの Copilot Chat は、社内情報を探して根拠にしながら答える体験に近い。

## 流れ

```text
社内の文書や会話
  ↓
質問する
  ↓
関連する社内情報を探す
  ↓
その情報をもとに回答
```

## 重要ポイント

- SharePoint 権限が効く
- OneDrive 権限が効く
- Teams、メール、会議、予定表も関係
- 専用RAGの前に検討する価値

## 注意点

```text
SharePoint 側の権限が雑
  ↓
Copilot に渡る前から雑
```

## 吹き出し

「既存の権限設計が、そのまま効いてくるのです…」
