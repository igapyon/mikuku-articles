# モデル名は、見えるものと見えないものがある

## 主題

Copilot はモデル固定の単体アプリではない。製品、地域、ライセンス、設定で見えるモデル名が変わる。

## 図解キーワード

- 見えるモデル名
- 系統だけ見える
- 未明示
- Smart モード
- Think Deeper
- Researcher agent
- Claude 系
- OpenAI 系
- 推測しない

## 対比

```text
見えている情報
  -> 表示されたモデル名や公式説明

見えていない情報
  -> 固定名で断定しない
```

## まとめ

「モデル名を暗記」より「公開範囲の違い」を見る。

## 吹き出し

「うぅ…見えていないモデル名は、言い切らないのが安全です…」
