# Microsoft 365 Copilot は、会社向けの業務AIとして見る

## 主題

Microsoft 365 Copilot は、会社の Microsoft 365 データとつながる業務AI。

## 接続するもの

- ファイル
- メール
- チャット
- 会議
- 予定表
- 人の情報

## 図解

```text
組織内の Microsoft 365 データ
  ↓
権限・管理者設定・セキュリティ
  ↓
Microsoft 365 Copilot
  ↓
仕事の文脈で回答
```

## 対比

```text
個人向けAIチャット
  -> 一般知識やWeb情報

Microsoft 365 Copilot
  -> 組織内データと仕事の文脈
```

## 吹き出し

「便利さと同じくらい、権限と運用が大事なのです…」
