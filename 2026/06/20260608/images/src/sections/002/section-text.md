# まずは大きく7つに分ける

## 主題

Copilot は 7 つの入口に分けると見通しがよくなる。

## 7分類

- Microsoft Copilot: 一般向けAIチャット
- Copilot in Microsoft 365: Officeアプリ内支援
- Microsoft 365 Copilot: 会社向け業務AI
- Microsoft 365 Copilot Chat: M365のチャット入口
- Copilot Studio: AIエージェント作成
- GitHub Copilot: 開発者向けAI
- Copilot Cowork: Frontierの実行エージェント

## 図解

```text
Copilot ファミリー
  ├─ 個人チャット
  ├─ Officeアプリ
  ├─ 会社の業務AI
  ├─ M365チャット
  ├─ 作る道具
  ├─ コード支援
  └─ 実行エージェント
```

## 吹き出し

「全部を一度に覚えなくても、入口を分ければ大丈夫です…」
