# Microsoft Copilot は、普通のAIチャットに近い

## 主題

Microsoft Copilot は個人でも使える一般向けAIチャットの入口。

## 主な入口

- copilot.com
- copilot.microsoft.com
- Windows
- macOS
- スマートフォン
- Edge

## 主な用途

- 質問
- 調べもの
- 文章下書き
- 要約
- 画像生成

## 注意点

```text
サインイン状態
個人アカウント / 職場・学校アカウント
Microsoft 365 契約
  ↓
使える機能や保護範囲が変わる
```

## 吹き出し

「同じ画面に見えても、サインイン状態で意味が変わることがあります…」
