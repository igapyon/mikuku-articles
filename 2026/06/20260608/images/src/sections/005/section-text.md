# Microsoft 365 アプリ内の Copilot は、Officeアプリの中に入る

## 主題

Word、Excel、PowerPoint、Outlook、Teams の中で、今開いている作業を助ける Copilot。

## 対象アプリ

- Word: 文章
- Excel: 表と分析
- PowerPoint: 資料
- Outlook: メール
- Teams: 会議とチャット

## 図解

```text
いつもの作業画面
  +
Copilot の補助席
  ↓
今開いている文書・表・メール・会議と結びつく
```

## 注意点

- アプリ内の機能
- 組織向け製品名としての Microsoft 365 Copilot
- 呼び方が重なるので分けて見る

## 吹き出し

「画面の中の機能なのか、組織向けの製品名なのかを分けます…」
