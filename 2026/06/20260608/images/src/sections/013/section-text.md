## 入口表

- 個人でAIチャット: Microsoft Copilot
- Word / Excel / PowerPoint / Outlook / Teams: Copilot in Microsoft 365
- 会社のメール・会議・ファイル: Microsoft 365 Copilot
- M365の安全なチャット入口: Microsoft 365 Copilot Chat
- 社内AIエージェント作成: Copilot Studio
- コードを書く: GitHub Copilot
- 今後の方向性を見る: Copilot Cowork
