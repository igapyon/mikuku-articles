# Copilot Cowork は、Frontier の先行公開系として見る

## 主題

Copilot Cowork は、Frontier プレビューの実験的・先行公開の実行エージェントとして慎重に見る。

## 方向性

```text
Copilot Chat
  -> 質問に答える / 要約する / 文章を作る

Copilot Cowork
  -> 仕事を受け取る / 計画する / 複数ステップを進める
```

## スキル

- Word
- Excel
- PowerPoint
- PDF
- Email
- Scheduling
- Meetings
- Enterprise Search
- Deep Research
- Agent Skills
- SKILL.md

## 注意点

- Frontier プレビュー
- 仕様変更の可能性
- 正式機能として断定しない

## 吹き出し

「答えるAIから、仕事を進めるAIへ近づく手がかりかもしれません…」
