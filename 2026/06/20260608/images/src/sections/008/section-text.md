# Edge から利用する Microsoft Search 体験では、社内検索に近づきやすい

## 主題

Edge for Business や Microsoft Search では、Web検索に近い導線で社内情報へ近づけることがある。

## 主な入口

- m365.cloud.microsoft
- SharePoint Online
- Edge for Business のアドレスバー
- Windows 検索ボックス

## 図解

```text
職場アカウントで Edge
  ↓
Web検索に近い操作
  ↓
Microsoft Search
  ↓
人・ファイル・SharePoint サイト
  ↓
アクセス権の範囲で表示
```

## 価値

- 社内システムを毎回選ばなくてよい
- 自然な入口
- 情報に近づきやすい

## 吹き出し

「Webを探す感覚で、社内情報に近づけることがあります…」
