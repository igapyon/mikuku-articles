# Copilot Studio は、Copilotを作る側の道具

## 主題

Copilot Studio は、使う側ではなく AIエージェントを作る側の道具。

## 作れるもの

- 社内FAQエージェント
- 業務手続き案内
- Teams で使うエージェント
- Webサイトから使うエージェント

## 対象者

- 業務システムを整える人
- Power Platform を使う人
- 社内AI活用を設計する人

## 対比

```text
まず使ってみたい
  -> Microsoft Copilot / M365アプリ内 Copilot

社内エージェントを作りたい
  -> Copilot Studio
```

## 吹き出し

「使う入口と、作る道具は分けて見るのです…」
