# セクション別グラレコ制作用整理テキスト

## 見出し

Prompt Engineering から Context Engineering へ

## 主題

よい指示を書くことに加えて、AIが迷わない状態を維持することが大事になる。

## 図解キーワード

- Prompt Engineering
- Context Engineering
- どうお願いするか
- 迷わない状態
- 途中で増えた情報
- 決まった方針
- 終わった作業
- 止める条件
- 小さな足場

## 対比

| Prompt Engineering | Context Engineering |
| --- | --- |
| 最初のお願い | 作業中の状態維持 |
| 言葉の工夫 | 文脈の置き場所 |
| 短い作業に強い | 長い作業を支える |

## 構図

```text
良い指示
  +
迷わない状態
  =
長いAIエージェント作業の足場
```
