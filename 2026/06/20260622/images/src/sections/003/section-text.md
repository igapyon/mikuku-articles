# セクション別グラレコ制作用整理テキスト

## 見出し

調べると、運用ファイルはいろいろある

## 主題

AIエージェント運用には多くのファイル候補があるが、最初から全部入れると重くなり、更新されないファイルが増える。

## 図解キーワード

- AGENTS.md
- PLAN.md
- WORKLOG.md
- RISKS.md
- ADR
- TODO.md
- プロジェクトルール
- 増やしすぎ注意
- 更新できる粒度
- Context Engineering

## 対比

| たくさん置く | 必要なものだけ置く |
| --- | --- |
| 圧倒される | 続けやすい |
| 更新されない | 品質を保ちやすい |
| コンテキストが濁る | 迷いにくい |

## 構図

```text
候補はいろいろ
  ↓
全部は重い
  ↓
必要な情報を必要な粒度で
```
