# セクション別グラレコ制作用整理テキスト

## 見出し

GOAL.md は、終わり方を決める

## 主題

`GOAL.md` はTODO一覧ではなく、何を達成したら終わりか、どんな時に止まるかを決める場所。

## 図解キーワード

- Objective
- Done
- Stop
- 完了条件
- 停止条件
- タスクリストにしない
- 同じ失敗3回で相談
- 終わり方
- 止まり方

## 対比

| GOAL.md に書く | 書きすぎない |
| --- | --- |
| 目的 | 細かい作業項目 |
| 完了条件 | 日々のTODO |
| 停止条件 | 作業ログ |

## 構図

```text
Objective
   ↓
Doneで完了確認
   ↓
Stopで無理に進まない
```
