# セクション別グラレコ制作用整理テキスト

## 見出し

まずは3ファイルから試してみたい

## 主題

最小構成として、目的・現在地・判断理由を3つのMarkdownへ分けて外部化する。

## 図解キーワード

- GOAL.md
- TODO.md
- DECISIONS.md
- 目的
- 現在地
- 判断理由
- 小さな足場
- 既存TODOはセクションで住み分け

## 役割分担

| ファイル | 役割 |
| --- | --- |
| GOAL.md | 終わり方を決める |
| TODO.md | 現在地を保つ |
| DECISIONS.md | 判断の理由を残す |

## 構図

```text
        GOAL.md
       目的・完了
          / \
         /   \
 TODO.md --- DECISIONS.md
 現在地       判断理由
```
