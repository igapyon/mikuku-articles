# リモート MCP は転送魔法陣

## 主題

リモート MCP は、手元を越えて遠くのサービスへお願いを届ける転送魔法陣。

## キーワード

- 遠くのサーバー
- 外部サービス
- 共有された道具
- network
- auth
- アクセス制御
- 権限設計

## 対比

| local MCP | remote MCP |
| --- | --- |
| 手元に近い | 遠くへ届く |
| 手触りが分かりやすい | 接続先が広がる |
| 範囲設計が大事 | 認証と権限が大事 |

## 図解

```text
手元の魔法陣
  -- network / auth -->
遠くの妖精さん
```

## 注意

どこへつながるのか、誰が使えるのか、どの範囲を許すのかを曖昧にしない。
