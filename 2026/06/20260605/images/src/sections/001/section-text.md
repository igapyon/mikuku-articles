# はじめに

## 主題

MCP を「妖精さんへお願いする魔法陣」として眺める入口。

## キーワード

- 硬い用語をやわらかく見る
- プロトコル / サーバー / ツール / リソース
- 会話の外へ手を伸ばす
- 小さな部室の机
- 魔法陣
- うさぎ型の妖精さんたち

## 図解

```text
難しい仕様
  ↓
まず比喩で輪郭をつかむ
  ↓
MCP = うさぎ型の妖精さんたちへお願いを届ける魔法陣
  ↓
あとで技術用語へ戻る
```

## 画面イメージ

- 魔法陣の周りに、小さなうさぎ型の妖精さんたちが数匹いる
- まだ CLI / API / database の細かな分類までは強調しない
- 「外の道具や資料へお願いを届ける相手がいる」雰囲気を出す
- 人型妖精ではなく、うさぎ型の小さなマスコットとして描く

## 伝えたいこと

最初から仕様を全部読むより、まず「お願いが届く形」という絵を持つ。
