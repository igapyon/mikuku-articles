# セクション別グラレコ制作用整理テキスト

## 主題

同じプロンプトを毎回書く手間を、魔法書からの呼び出しに変える。

## キーワード

- 毎回手打ち
- 同じ説明
- 長い詠唱
- 保存
- 呼び出し
- 詠唱省略

## 図解案

```text
毎回プロンプトを書く
  ↓
同じ説明を何度もする
  ↓
長い詠唱になる
  ↓
魔法書に書いておく
  ↓
必要なときに呼び出す
```

## 対比

| Before | After |
| --- | --- |
| その場で書く | Skill から呼び出す |
| 説明が長い | 詠唱を省略 |
| ばらつく | 安定する |

## 画像内の短い言葉

- 「文体」
- 「Markdown」
- 「余計な情報を足さない」
- 「温度感」
- 「事実確認」
