# セクション別グラレコ制作用整理テキスト

## 主題

Agent Skills は、プロンプトや資料を束ね、育てていける作業の魔法書。

## キーワード

- まとめ
- 束ねる
- 呼び出す
- 術式
- 禁則事項
- MP効率
- 育てる

## 図解案

```text
プロンプト / 資料
  ↓
呪文として呼び出す
  ↓
術式と禁則事項で整える
  ↓
MP効率を考える
  ↓
自分の作業に合う魔法書
```

## 最終メッセージ

- 魔法ではない
- でも魔法書みたいに育てられる
- 少しずつ書き足し、整える

## 画像内の短い言葉

- 「作業の中で効果になる」
- 「手触りに近い比喩」
- 「素敵な魔法書を綴る」
