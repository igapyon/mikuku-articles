# セクション別グラレコ制作用整理テキスト

## 主題

Agent Skills も MP のような利用枠を消費する。厚すぎる魔法書は開くだけで重くなる。

## キーワード

- MP
- トークン
- 利用枠
- 分厚すぎる魔法書
- 唱えきれない
- 効果がぼやける

## 図解案

```text
分厚い魔法書
  ↓
読むだけで MP 消費
  ↓
必要な呪文に届きにくい
  ↓
肝心の効果がぼやける
```

## 対比

| 充実 | 詰め込みすぎ |
| --- | --- |
| 必要な情報がある | 入口で疲れる |
| 効果が安定する | 読み込みが重い |
| 呪文に届く | MP不足 |

## 画像内の短い言葉

- 「MP不足」
- 「途中で息が足りない」
- 「古い注意書き」
- 「めったに使わない補足」

## みくく吹き出し

「唱えたいのに、MPが足りないと開ききれないのです…」
