# セクション別グラレコ制作用整理テキスト

## 主題

魔法書には、限られた MP で必要な呪文に届くための整理技が必要。

## キーワード

- MP効率
- 章立て
- 入口
- 奥のページ
- 短縮詠唱
- 必要時だけ開く

## 図解案

```text
探しやすい章立て
  + よく使う呪文は入口
  + 大技は奥のページ
  + 長い呪文は短縮
  ↓
MP効率アップ
```

## 対比

| 整理前 | 整理後 |
| --- | --- |
| 本棚の前で迷う | 必要な呪文へ直行 |
| 毎回全部読む | 必要時だけ開く |
| 効果がぼやける | 効果が届く |

## 画像内の短い言葉

- 「入口近く」
- 「奥のページ」
- 「短縮詠唱」
- 「毎回 / 必要時」

## みくく吹き出し

「強い呪文を増やすだけでなく、探しやすくするのです…」
