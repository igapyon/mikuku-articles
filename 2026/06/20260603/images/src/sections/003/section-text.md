# セクション別グラレコ制作用整理テキスト

## 主題

魔法書には名前だけでなく、使う場面や条件を決める「術式」がある。

## キーワード

- 術式
- 使う場面
- 使わない場面
- 条件
- 材料
- 効果の調整

## 図解案

```text
同じ生成AI
  + 立場
  + 丁寧さ
  + 推測範囲
  + 未確認の扱い
  ↓
出力の表情が変わる
```

## 対比

| 呪文だけ | 術式つき |
| --- | --- |
| 「書いて」だけ | 目的と条件がある |
| 効果がぶれる | 目的の形に収まる |
| 強さ重視 | 制御重視 |

## 画像内の短い言葉

- 「目的の形に収める」
- 「小さな条件」
- 「出力の表情」
- 「業務メモ / 技術記事 / レビュー」

## みくく吹き出し

「強くするだけではなく、形を整えるところが大事なのです…」
