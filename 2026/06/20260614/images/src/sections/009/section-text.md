# 有識者レビューで発掘結果を照合する

## 主題

仕様復元は AI agent だけで完結させない。AI agent の仮説を有識者が照合し、抜けや誤り、業務上の意味を補う。

## レビューサイクル

```text
AI agent が作る
  ↓
有識者が見る
  ↓
抜け・誤り・業務意味・運用注意を補う
  ↓
AI agent が反映
  ↓
再解析
```

## レビューで得るもの

- ここはこういう意味
- 今は使っていない
- この項目はこの業務で使う
- ソースコードには出ない運用前提

## 注意点

- 全体仕様を最初から聞くのは重い
- 一覧や仕様メモを見ながら少しずつ聞く
- 有識者が離職すると困難になる

## 吹き出し案

「見てもらうことで、ようやく思い出せる仕様もあります…」
