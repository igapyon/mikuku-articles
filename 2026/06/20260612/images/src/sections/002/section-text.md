# 002 「心」を論じない方の心理学

## 主題

- 心の中を読むのではなく、観察できるふるまいを見る
- 生成AIとの対話も、入力と出力の関係として観察できる

## 図解キーワード

- ふるまい
- 反応
- 会話
- 選択
- 行動の変化
- 心を読まない
- 会話の型を見る

## 観察ポイント

```text
強く言う      -> 謝る？
具体的に言う  -> 検証する？
反論を許す    -> 留保する？
返事を返す    -> 次が変わる？
```

## 対比

| 心を読む見方 | 型を見る見方 |
| --- | --- |
| 内面を推測 | 応答を観察 |
| 感情を断定 | 入出力を見る |
| 結論が重い | 実務に使える |

## 吹き出し案

「心を決めつけずに、返ってくる型を見ます…」
