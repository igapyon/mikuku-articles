# 010 人間に話すときのように、でも人間だと決めつけずに

## 主題

- 生成AIに心があると決めつけない
- でも、人間に話すときのように言葉を整えると、能力を引き出しやすい
- 礼儀だけではなく、共通基盤を整える実務

## 図解キーワード

- 人間同士の会話の型
- 言葉を整える
- 依頼
- 指摘
- 修正点
- 共通基盤
- 関心領域
- 実務

## バランス

```text
人間だと信じる必要はない
  +
人間に話すように整える
  =
より良い会話設計
```

## 対比

| 決めつけ | 実務として扱う |
| --- | --- |
| AIには心がある | 心の有無とは別 |
| 礼儀だけの話 | 出力品質の話 |
| 道具だから雑でよい | 言い方が応答を変える |

## 吹き出し案

「人間だと決めつけず、でも言葉は整えるのです…」
