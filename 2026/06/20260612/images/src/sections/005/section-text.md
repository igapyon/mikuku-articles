# 005 謝罪は、内面といつも一致するとは限らない

## 主題

- 生成AIの謝罪を「本当に反省した」と読まない
- ただし、人間の謝罪も常に内面と一致するわけではない
- 大事なのは、謝罪のあと何が変わったか

## 図解キーワード

- 謝罪
- 場を収める
- 関係を守る
- 応答パターン
- 検証
- 修正
- 前提確認

## 判断ポイント

```text
謝罪した
  ↓
理解した？ ではなく
  ↓
何が変わった？
  ↓
前提確認 / 修正方針 / 再発防止の観点
```

## 対比

| 見ない | 見る |
| --- | --- |
| 謝罪の言葉だけ | その後の変化 |
| 反省の有無 | 検証と修正 |
| 感情の推測 | 方針の具体化 |

## 吹き出し案

「謝罪より、その後の検証を見るのです…」
