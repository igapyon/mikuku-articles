# 003 生成AIとのうまい会話に使えそうな、いくつかの見方

## 主題

- プロンプト技術だけでなく、会話の進め方が出力に影響する
- 心や感情を外し、対話を扱いやすくする見取り図を作る

## 5つの見方

```text
行動分析   = 入力と出力を見る
会話分析   = ずれを修復する
共通基盤   = 前提をそろえる
発話行為   = 言葉を操作として見る
社会心理学 = 圧・迎合・同調に注意
```

## 図解の中心

- 入力と出力
- ずれの修復
- 前提共有
- 言葉の操作
- 反論と留保の余地

## 対比

| うまくいきにくい | うまくいきやすい |
| --- | --- |
| 一発で当てようとする | ずれを直す |
| 黙って期待する | 前提を言葉にする |
| 従わせる | 反論も許す |

## 吹き出し案

「会話を整える見取り図として使うのです…」
