# 006 糾弾して従わせると、よい応答は出にくい

## 主題

- 強い言葉で詰めると、生成AIは謝罪・同意・迎合に寄りやすい
- 反論、留保、確認、代替案が弱くなることがある
- 良い仕事には、考える余地が必要

## 図解キーワード

- 強い圧
- 謝罪
- 同意
- 迎合
- 低姿勢
- 反論が弱い
- 考える余地

## 流れ

```text
強く責める
  ↓
謝罪・迎合に寄る
  ↓
反論や留保が弱くなる
  ↓
ベストパフォーマンスが出にくい
```

## 対比

| 従わせる会話 | 考えさせる会話 |
| --- | --- |
| だめ、ちゃんとやって | どこが違うか分ける |
| 同意に寄る | 確認できる |
| 反論しにくい | 代替案が出る |

## 吹き出し案

「従わせるだけだと、反論が消えてしまうかも…」
