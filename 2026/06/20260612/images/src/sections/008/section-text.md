# 008 フィードバックは、次の応答への入力になる

## 主題

- 生成AIとの会話では、短い反応も次の応答の手がかりになる
- フィードバックは感情的な儀礼ではなく、品質を上げる入力

## 図解キーワード

- いいね
- そこは違う
- ありがとう
- 方向が合っている
- 修正が必要
- 区切り
- 次の応答

## 流れ

```text
応答を受け取る
  ↓
短いフィードバック
  ↓
方向が伝わる
  ↓
次の応答が変わる
```

## 対比

| 省略する | 返す |
| --- | --- |
| 意図を推測させる | 方向を示す |
| ずれが残る | 修正しやすい |
| 会話が流れる | 共通理解が育つ |

## 吹き出し案

「小さな返事も、次の応答への入力なのです…」
