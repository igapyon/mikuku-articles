# 001 はじめに

## 主題

- 生成AIは人間っぽく見えることがある
- でも、この記事では心の有無を論じない
- 見るのは会話上の入力と出力

## 図解キーワード

- 心を論じない
- 入力と出力を見る
- 強く責める
- 問題点を分ける
- 反論を許す
- 応答の型

## 流れ

```text
人間の言葉
  ↓
生成AIの返答
  ↓
謝罪 / 迎合 / 検証 / 留保
  ↓
会話の力学を観察
```

## 対比

| 踏み込まない問い | 観察する問い |
| --- | --- |
| 心があるか | どう返すか |
| 意識があるか | 何に寄るか |
| 感情があるか | 修正できるか |

## 吹き出し案

「あ、あの…まずは心ではなく、会話の入出力を見るのです…」
