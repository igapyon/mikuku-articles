# 004 発話行為として見る：言葉が何をしているかを見る

## 主題

- 言葉は情報を渡すだけではなく、会話を動かす操作にもなる
- ありがとう、違うよ、この方向で続けて、などは次の応答を形づくる

## 図解キーワード

- 言葉の操作
- ありがとう
- 違うよ
- この方向で続けて
- 区切り
- 修正信号
- 探索範囲
- 会話を整える

## 対応

| 言葉 | 役割 |
| --- | --- |
| ありがとう | 区切り・役立った信号 |
| 違うよ | 修正信号 |
| この方向で続けて | 探索範囲を狭める |

## 吹き出し案

「言葉は、会話を動かす小さな操作なのです…」
