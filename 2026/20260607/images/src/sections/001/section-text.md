# セクション001 グラレコ制作用整理テキスト

## 主題

- 生成AIの関心が変わる
- 「できるか」だけでなく「何トークンでできるか」
- 省トークン = 短くするだけではない

## 図解キーワード

- 実行時トークン効率
- 入力 / 出力 / cached input / reasoning tokens
- 小さな差が繰り返しで積み上がる
- 開発時効率と実行時効率を分ける

## 中心図

```text
何ができるか
  ↓
何トークンで安定実行できるか
```

```text
開発時の効率
  作るまでの試行錯誤

実行時の効率
  1回動かすたびの消費
```

## 伝えたい一文

- 省トークンは、毎回渡す意味を置き直す設計。
