# 会話の外へ手を伸ばす MCP

## 主題

生成AIとの会話だけでは届かない外部の資料や道具へ、お願いを届ける入口。

## キーワード

- 会話の中でできること
- 会話の外にあるもの
- 手元の資料
- 外の道具
- 別の場所の記録
- お願いの通り道

## 対比

| 会話だけ | MCP が必要 |
| --- | --- |
| 文章を考える | ファイルを参照する |
| 方針を整理する | 道具を動かす |
| 考えを言葉にする | 記録を確認する |

## 図解

```text
生成AI agent
  ↓ 会話だけでは届かない
外部資料 / 道具 / 記録
  ↓
MCP がお願いの形を用意する
```
