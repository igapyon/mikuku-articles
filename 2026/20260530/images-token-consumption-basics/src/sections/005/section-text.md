# 005 コンテキストウィンドウとは何か

## 主題

コンテキストウィンドウは、生成AIが一度に扱える文脈の範囲。

## キーワード

- 文脈
- 一度に扱える範囲
- 会話履歴
- 指示
- 資料
- 上限

## 図解

```text
コンテキストウィンドウ
  ├ 現在の指示
  ├ 会話履歴
  ├ 資料
  └ 参照情報
```

## 対比

| 誤解 | 実際 |
| --- | --- |
| 無限に入る箱 | 上限がある文脈枠 |
| 大きいほど常に良い | 入力トークンも増えやすい |
| 入れた情報は無料 | 処理量になりうる |

## みくく吹き出し案

「便利な箱なのですが、無限ではないのです…」
