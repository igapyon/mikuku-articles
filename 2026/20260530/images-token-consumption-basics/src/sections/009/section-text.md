# 009 まず見るところ

## 主題

トークン消費量を考えるとき、最初に見るのは入力量、出力量、重複処理。

## キーワード

- 入力量
- 出力量
- 重複処理
- 短く答えて
- 要点だけ
- 必要な資料だけ

## 3つの確認点

```text
1. 入力として渡している文章量
2. 出力として返させている文章量
3. 同じ情報を何度も処理に乗せていないか
```

## 軽くするヒント

- 回答を短く指定する
- 要点だけにする
- 関係しそうなファイルを全部読ませない
- 過去の会話を引きずりすぎない
- 必要な前提だけまとめる
