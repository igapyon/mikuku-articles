# 003 トークンとは何か

## 主題

トークンは、AI が文章を扱うための小さな部品。文字数や単語数とは一致しない。

## キーワード

- 小さな部品
- tokenizer
- 文字数ではない
- 単語数ではない
- モデルごとに違う

## 例

```text
生成AIを使う
  ↓
生成 / AI / を / 使う

tokenization
  ↓
token / ization
```

## 対比

| 誤解 | 実際 |
| --- | --- |
| 100文字 = 100トークン | そうとは限らない |
| 1単語 = 1トークン | 複数に分かれることがある |
| どのモデルでも同じ | tokenizer によって違う |

## 覚えること

- 長い入力ほど入力トークンは増えやすい
- 長い回答ほど出力トークンは増えやすい

## みくく吹き出し案

「トークンは、文字数でも単語数でもないのです…」
