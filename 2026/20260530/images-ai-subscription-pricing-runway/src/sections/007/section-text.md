# セクション整理: Agent Skills で、利用を蓄積に変える

## 主題

Agent Skills に作業手順や文体や判断基準を残すと、生成AI利用がその場限りではなく蓄積になる。

## グラレコ化キーワード

- Agent Skills
- 作業手順
- 文体
- 判断基準
- 小さな足場
- 前回の工夫
- 次の作業
- 循環
- ポジティブなスパイラル

## 循環図

```text
使う
  ↓
うまくいった手順を残す
  ↓
Skill にする
  ↓
次の作業が速くなる
  ↓
新しい工夫が見つかる
```

## 例

- 記事の文体 skill
- リポジトリ規約 skill
- レビュー観点 skill
- CLI 操作手順 skill

## 吹き出し案

「前回の工夫が、次の小さな足場になるのです…」
