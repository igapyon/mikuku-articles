# セクション整理: 昔から作りたかったアプリを、今のうちに形にする

## 主題

実装の重さが下がった今、積み残していたアイデアを実際のアプリへ変えやすい。

## グラレコ化キーワード

- いつか作りたい
- 小さなツール
- 自分用 Web アプリ
- 趣味の音楽や文章
- 仕事の前処理
- 設計、実装、画面、テスト、ドキュメント、配布
- 人間が芯を持つ
- 完成品と経験値

## 図解構造

```text
積み残したアイデア
  ↓ AI と分解
設計 / 実装 / テスト / 文書化
  ↓
小さく完成
  ↓
作る間合いが残る
```

## 残るもの

- 完成したアプリ
- AI への説明の仕方
- レビューすべき場所
- Agent Skills
- 開発メモ
- テスト

## 吹き出し案

「完成品と経験値が、同時に残るのです…！」
