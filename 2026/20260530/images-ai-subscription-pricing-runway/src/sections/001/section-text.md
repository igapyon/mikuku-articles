# セクション整理: はじめに

## 主題

生成AIは、なぜ月額サブスクや無料枠で高性能に使えてしまうのか。

## グラレコ化キーワード

- 親しみやすい価格
- 高性能AI
- 月額サブスク
- 無料枠
- 文章、コード、調査、設計
- 裏側の計算資源
- GPU / TPU
- データセンター
- 電力
- 研究者、エンジニア、運用

## 図解構造

```text
ユーザー画面
  ↓
便利な生成AI
  ↓
見えない裏側
  ↓
巨大な計算資源
```

## 対比

| 表側 | 裏側 |
| --- | --- |
| 月額で使える | 計算資源が必要 |
| 気軽に質問 | 推論コスト |
| 便利な画面 | データセンター |
