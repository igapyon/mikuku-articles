# セクション整理: おわりに

## 主題

いまは生成AIを眺めるだけではなく、会話し、作り、癖をつかみ、手順を Skill にしていく時期。

## グラレコ化キーワード

- 親しみやすい価格
- 戦略価格
- 競争
- 法人契約 / API 収益
- 先行投資
- 特別な時期
- AI agent とコードを書く
- 作りたかったものを形にする
- Agent Skills
- 作業能力を育てる

## まとめ図

```text
月額で試せる
  ↓
AI と作る
  ↓
癖をつかむ
  ↓
手順を Skill 化
  ↓
次の作業能力が育つ
```

## 強調メッセージ

- ただ眺める時期ではない
- 今の助走期間に何を作るか
- 手元の行動が未来の作業能力になる

## 吹き出し案

「今この助走期間に何を作るかは、まだ手元にあります…！」
