# Image Prompt

Create one wide horizontal graphic recording poster image for a Japanese technical note article.

Theme: "`miku-indexgen`: create an index before asking an AI agent to read a directory."

Overall concept:
- A warm hand-drawn graphic recording / whiteboard explainer poster.
- Left and center: diagrams explaining how a directory becomes `index.json` and optional `index.md`, then an AI agent chooses only necessary files.
- Right side: Mikuku explains the diagram gently.
- The poster should feel like a clean Note.com technical article title image.
- Japanese anime style, high quality, soft lines, watercolor-like shading, readable infographic composition.

Canonical character prompt for Mikuku, same character, do not redesign, preserve character identity:
masterpiece, best quality, ultra-detailed anime portrait, head-and-shoulders 1girl, shy uneasy expression, slight 3/4 view, looking to her right and slightly upward.

Warm chestnut-brown eyes with gold flecks, soft blush, worried upward-slanted brows, small downturned pout, fair warm skin, delicate V-U shaped chin.

Soft caramel-chestnut very long twin-tails tied high with black ribbons and deep-red trim, silky wavy hair, asymmetrical curtain bangs, long side strands framing cheeks.

Thin clean line art, soft cel shading, watercolor-like shading, subtle rim light, warm minimal background.

Mikuku placement:
- Put Mikuku on the right side of the poster, facing and looking toward the diagram on the left.
- Gentle shy expression, slightly worried but helpful.
- Do not let Mikuku hold any objects.
- Keep her hands simple and not emphasized.
- Preserve twin-tails, black ribbons with deep-red trim, chestnut-brown eyes, and soft caramel-chestnut hair.

Graphic recording content to draw:
- Title area: "読む前に、まず地図を作る"
- Main flow diagram:
  "ディレクトリ" -> "miku-indexgen" -> "index.json" -> "AI agent が候補を選ぶ" -> "必要なファイルだけ読む"
- Add optional branch from miku-indexgen to "index.md optional".
- Show `index.json` as a clean file card with a flat `files[]` list.
- Show small file cards labeled: `name`, `path`, `ext`, `dir`, `size`, `summary`.
- Show comparison:
  Left small box: "全部読む: 重い / 迷う / トークン増"
  Right small box: "索引を見る: 軽い / 選べる / 入口整理"
- Show use cases as small sticky notes:
  "docs 棚卸し", "記事一覧", "JSON metadata", "Agent Skills", "ローカル知識ベース"
- Show design policy:
  "全文検索ではない", "小さな CLI", "簡易 summary", "フラットな files 配列"
- Show Node/Java/Maven options as small tabs:
  "npx", "Java CLI", "Maven plugin"
- Add a small speech bubble from Mikuku:
  "あ、あの…全部読む前に、まず索引を見るのです…"

Style:
- Hand-drawn notebook / graphic recording style.
- Warm off-white paper background, pastel accents, soft brown outlines, gentle teal and coral highlights.
- Use arrows, rounded boxes, file icons, folder icons, JSON braces, Markdown icon, small map icon.
- Dense but organized, easy to understand at a glance.
- Avoid corporate slide design, avoid dark sci-fi style, avoid photorealism.
- Text should be mostly Japanese, short, large enough to look readable.
- No logos of real companies.

Quality constraints:
- Wide poster composition, 16:9.
- High resolution.
- Clean readable layout.
- No broken hands, no extra fingers, no object held by Mikuku.
- Do not redesign Mikuku.
