# Attention グラレコ制作用整理テキスト

## 主題

Attention は、今のトークンを考えるときに文脈のどこを見るかを決める仕組み。

## 図解キーワード

- どこを見るか
- 参照の強さ
- Query
- Key
- Value
- 文脈を混ぜる

## 例

```text
太郎は花子に本を渡した。彼女はそれを読んだ。

彼女 -> 花子
それ -> 本
```

## QKV

| 名前 | 感覚 |
|---|---|
| Query | 何を探すか |
| Key | 照合される目印 |
| Value | 受け取る中身 |

## みくく吹き出し

「Attention は、文脈の中でどこに目を向けるかの仕組みです…」
