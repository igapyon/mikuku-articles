# 文章はトークンに分けられる グラレコ制作用整理テキスト

## 主題

モデルは文章をそのまま読むのではなく、トークンという小さなかけらに分ける。

## 図解キーワード

- token
- 文章のかけら
- モデルが読む単位
- 単語とは限らない
- tokenizer

## 流れ

```text
今日は猫が寝ている
  ↓
今日は / 猫 / が / 寝て / いる
```

## 注意点

- 実際の分割はモデルや tokenizer で違う
- 日本語、英語、記号、空白も対象
- モデルは最初から「文」や「意味」をそのまま持たない

## みくく吹き出し

「トークンは、モデルが文章を読むときの小さなかけらです…」
