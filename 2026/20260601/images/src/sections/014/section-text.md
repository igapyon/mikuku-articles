# 学習と利用は同じではない グラレコ制作用整理テキスト

## 主題

通常の会話中にモデル本体の重みがその場で更新されるわけではなく、主に context が変わる。

## 図解キーワード

- 学習
- 会話 context
- メモリ
- RAG
- fine-tuning
- 一時的な文脈

## 対比表

| 種類 | 何が変わるか | いつ効くか |
|---|---|---|
| 学習 | モデルの重み | 学習後のモデル全体 |
| 会話 context | 入力や履歴 | その会話の範囲 |
| メモリ / RAG | 外部情報 | 参照されたとき |
| fine-tuning | 追加学習された重み | 調整後 |

## みくく吹き出し

「さっき教えたから覚えた、に見えても context の効果かもしれません…」
