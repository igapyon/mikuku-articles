# 014 重い処理は scripts や CLI や MCP に切り出す

## 主題

定型的で重い処理は、長い自然言語手順ではなく、実行できる道具へ切り出せる。

## 重要キーワード

- `scripts/`
- CLI
- MCP server
- 長い手順説明
- 定型処理
- 入出力が安定
- 実行型
- ハイブリッド型
- コンテンツ型

## Before / After

```text
長い自然言語手順
  ↓
短いコマンド案内
```

## 判断

- 処理が揺れている → 自然言語が扱いやすい
- 入出力が安定 → scripts / CLI / MCP が向く
- 静的コンテンツ中心 → 無理に scripts は不要

## 吹き出し案

「毎回説明するより、道具に任せられることもあります…」
