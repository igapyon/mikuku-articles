# 005 小さなスキルに分ける

## 主題

巨大な万能 Skill は便利に見えるが、関係の薄いルールや参照まで持ち込みやすい。

## 重要キーワード

- 巨大な万能 Skill
- 用途別の小さな Skill
- 発動条件
- 参照範囲
- 成果物の種類
- 禁止事項
- 品質基準
- 選択コスト

## Before / After

```text
全部入り Skill
  ↓
用途別の小さな Skill
```

## 分ける例

- 記事執筆
- レビュー
- 画像生成・グラレコ
- GitHub Release / PR
- 特定プロジェクト開発

## 注意

細かく分けすぎると、どれを使うか迷いやすい。

## 吹き出し案

「必要な子だけ呼べる大きさがよさそうです…」
