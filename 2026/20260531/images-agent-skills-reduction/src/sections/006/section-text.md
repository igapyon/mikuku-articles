# 006 description が発火精度を左右する

## 主題

`description` は Skill をいつ使うかを伝える短い判定材料で、発火精度に大きく関わる。

## 重要キーワード

- `description`
- `name`
- 発火精度
- 発火の引き金
- 使う条件
- 使わない条件
- 明示されたときだけ
- 広すぎず狭すぎず

## 対比

| 広すぎる | 狭すぎる | ちょうどよい |
| --- | --- | --- |
| 関係の薄い場面で発火 | 必要な場面で発火しない | 必要な場面で自然に発火 |
| 文脈が増える | 言い直しが増える | 軽く正確に始まる |

## 図解

```text
ユーザー依頼
  ↓
name + description
  ↓
Skill を使うか判定
```

## 吹き出し案

「短い description でも、責任は大きいのです…」
