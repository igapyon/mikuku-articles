# 017 おわりに

## 主題

Agent Skills のトークン節約は、情報を減らすことではなく、どの入口からどの順序で情報へたどり着くかを設計すること。

## 重要キーワード

- 不要な Skill を発火させない
- 適切な粒度
- `description`
- `SKILL.md`
- `references/`
- `templates/`
- `examples/`
- `index.json`
- front matter
- `distilled/`
- scripts / CLI / MCP

## まとめの流れ

```text
発火を絞る
  ↓
SKILL.md を入口にする
  ↓
詳しい知識を分ける
  ↓
候補を絞る
  ↓
蒸留する
  ↓
定型処理は外へ出す
```

## 最終メッセージ

情報は残す。でも、毎回全部は読ませない。
