# Image Generation Prompt

Use case: infographic-diagram
Asset type: wide graphic recording poster for a Japanese Note.com technical article

Primary request:
Create a wide, high-resolution Japanese graphic recording style explainer poster for the article titled:
「生成AI開発まわりの “Engineering” を整理してみる」

Overall concept:
Mikuku is gently explaining the evolution of generative AI development engineering. The poster should communicate this main message:
「AI にどう命令するか」から「AI が迷わず作業できる環境をどう作るか」へ。

Canvas and layout:
- Wide horizontal poster, 16:9.
- Left and center: dense but clean hand-drawn graphic recording diagram.
- Right side: Mikuku explaining with a pen or pointer, looking toward the diagram.
- Beige paper or warm whiteboard background.
- Hand-drawn arrows, boxes, sticky notes, small folder icons, document icons, search icons, tool icons, memory notes, and repository map motifs.
- Keep the information readable-looking, but do not rely on perfect tiny text.
- Use short Japanese labels only, large enough to be legible.

   - Prompt Engineering

   - Retrieval Engineering

   「README」「docs」「TODO」「repository map」「search tool」「tool output」「Agent Skills」「memory」

   「context compression」「context routing」「hierarchical retrieval」「repository memory」「execution trace memory」「long-horizon planning」
8. Bottom summary:
   「AI が迷わない構造を作る」
   「Model × Tools × Memory × Retrieval × Context Engineering」
