# 001 はじめに

## 主題

生成AIを使ううえで「トークン」という見方を知る入口。

## キーワード

- トークン
- 生成AIの動き方
- AI agent
- API
- 入力
- 出力
- コンテキストウィンドウ
- 利用量

## 図解の流れ

```text
ふつうのチャット
  ↓
AI agent / API を使う
  ↓
トークンが前面に出る
  ↓
入力・出力・文脈・利用量が見えやすくなる
```

## 伝えたいこと

- 質問、会話履歴、資料、回答はトークンとして扱われる
- トークンを知ると、生成AIの裏側が少し見える
- この記事は、入力、出力、コンテキスト、利用量への入口
