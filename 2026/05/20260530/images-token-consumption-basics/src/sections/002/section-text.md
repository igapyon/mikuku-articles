# 002 まず、生成AIはトークンで動いている

## 主題

生成AIは、文章をトークンに変換し、次に来そうなトークンを計算して応答を作る。

## キーワード

- 文章
- トークン列
- 次のトークン
- 応答生成
- 根っこの仕組み

## 図解の流れ

```text
人間の文章
  ↓
トークンの並び
  ↓
次に来そうなトークンを計算
  ↓
応答文になる
```

## 役割

| 要素 | 役割 |
| --- | --- |
| ユーザーの質問 | 入力として分解される |
| 会話履歴 | 文脈として分解される |
| 読ませた資料 | 入力情報として分解される |
| AIの回答 | 出力トークンとして作られる |

## みくく吹き出し案

「生成AIのかなり根っこのところに、トークンがあるのです…」
