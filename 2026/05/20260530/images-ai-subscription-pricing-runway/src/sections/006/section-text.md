# セクション整理: 使い倒すとは、会話量を経験値に変えること

## 主題

使い倒すとは、質問数を増やすことではなく、AI との会話から作業の勘所を得ること。

## グラレコ化キーワード

- 会話量
- 経験値
- 任せる粒度
- 失敗する境界
- 人間が設計を握る場所
- AI agent が作業しやすい構成
- テスト
- 確認手順
- 説明を残す

## 図解構造

```text
頼む
  ↓
失敗する
  ↓
直す
  ↓
癖をつかむ
  ↓
次がうまくなる
```

## 対比

| 量だけ | 経験値化 |
| --- | --- |
| たくさん質問 | 境界を知る |
| 回答を消費 | 手順を残す |
| その場で終わる | 次回に活かす |

## 吹き出し案

「どこまで任せるかを、手を動かして覚えるのです…」
