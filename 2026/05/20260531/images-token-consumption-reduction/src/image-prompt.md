# Image Generation Prompt

Create a high-resolution horizontal graphic recording poster for a Japanese technical article.

## Core Theme

Japanese title text in the poster:

「生成AIのトークン消費量を抑える考え方」

Main message:

「短くするだけでなく、AIに渡す文脈を選ぶ」

The image should explain how to reduce token consumption in daily generative AI and AI agent usage. Make it feel like a warm hand-drawn graphic recording board for a Note.com technical essay.

## Layout

- Wide horizontal poster, 16:9.
- Left and center: hand-drawn infographic / graphic recording.
- Right side: Mikuku explains the board, looking toward the diagram.
- Mikuku must not hold any object.
- Use short readable Japanese labels, not long paragraphs.
- Use arrows, boxes, small icons, notes, folders, markdown symbols, speech bubbles, and a simple cycle diagram.
- Clean margins, readable hierarchy, warm off-white paper background.

## Visual Structure To Draw

Center title block:

「トークン節約 = 文脈を選ぶ」

Flow diagram:

```text
目的を小さく
  ↓
入力を選ぶ
  ↓
履歴を整理
  ↓
出力を指定
  ↓
agent は絞って読む
  ↓
情報は外へ
  ↓
要点を蒸留
```

Contrast area:

```text
重い使い方
- 全部読んで
- 資料まるごと
- 長いログ全部
- 毎回長い説明

軽い使い方
- この観点だけ
- 必要な章だけ
- 差分だけ
- 共通情報はファイルへ
```

Important warning box:

「削ってはいけない情報」

Include six small chips:

「目的」「制約」「判断基準」「禁止事項」「成果物の形式」「重要な前提」

AI agent mini diagram:

```text
ファイル一覧・検索
  → 候補を絞る
  → 必要箇所だけ読む
```

External memory / distillation mini diagram:

```text
Markdown / JSON / TODO
  → 短い入口メモ
  → 次回が軽い
```

Session switch note:

「長い会話は、必要な文脈だけ持って新しいセッションへ」

Footer keyword:

「Context Engineering」

## Mikuku Speech Bubble

Use one speech bubble near Mikuku:

「あ、あの...短くするより、何を見せるかを選ぶのが大事です...！」

## Mikuku Canonical Character Prompt

Use this as the canonical character prompt for Mikuku. Same character, do not redesign, preserve character identity, canonical character reference. Preserve face shape, hairstyle, hair color, twin-tails, hair ribbons, eyes, shy gentle personality impression, and soft Japanese anime technical explainer atmosphere.

masterpiece, best quality, ultra-detailed anime portrait, head-and-shoulders 1girl, shy uneasy expression, slight 3/4 view, looking to her right and slightly upward.

Warm chestnut-brown eyes with gold flecks, soft blush, worried upward-slanted brows, small downturned pout, fair warm skin, delicate V-U shaped chin.

Soft caramel-chestnut very long twin-tails tied high with black ribbons and deep-red trim, silky wavy hair, asymmetrical curtain bangs, long side strands framing cheeks.

Thin clean line art, soft cel shading, watercolor-like shading, subtle rim light, warm minimal background.

For this poster, show Mikuku on the right side as a half-body or upper-body anime character, not just a portrait, gently facing the diagram. Keep the same character. Do not let Mikuku hold any objects.

## Style

- Japanese anime style, cute but professional.
- Graphic recording illustration.
- Educational infographic poster.
- Hand-drawn notebook style.
- Warm off-white paper, soft brown linework, accent colors in teal, coral, mustard, and gentle green.
- Avoid a one-color palette.
- High detail, clean, bright, readable.
- No photorealism, no dark sci-fi mood, no corporate slide look.
- Avoid tiny dense text; use short labels.
- No extra fingers, no broken hands, no objects in Mikuku's hands.
