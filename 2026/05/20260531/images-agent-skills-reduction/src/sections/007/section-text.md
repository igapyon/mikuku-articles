# 007 SKILL.md を太らせすぎない

## 主題

`SKILL.md` は本文ではなく、入口と索引にする。

## 重要キーワード

- `SKILL.md`
- 入口
- 索引
- 発火条件
- 禁止事項
- 成果物の形式
- 参照順序
- 長い背景説明
- 長い例文

## 残すもの

- いつ発火するか
- いつ発火しないか
- 最初に読む参照
- 成果物の形式
- 禁止事項
- 品質境界
- 運用ルール

## 外へ出すもの

- 長い背景説明
- 詳細な判断基準
- テンプレート本文
- 機械的な処理手順

## 吹き出し案

「入口で全部説明しすぎないのが大事です…」
