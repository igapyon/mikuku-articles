# 001 はじめに

## 主題

Agent Skills は、知識や手順を追加するだけでなく、毎回全部読ませないための仕組みでもある。

## 重要キーワード

- Agent Skills
- トークン消費量
- 知識を足す
- 毎回全部読ませない
- `SKILL.md`
- `references/`
- `examples/`
- `templates/`
- どの情報をいつ読ませるか

## 図解ポイント

```text
Agent Skills
  = 知識を足す
  + 毎回全部読ませない
```

## 流れ

```text
一般的な節約術
  ↓
Agent Skills に絞る
  ↓
情報の出会わせ方を設計する
```
