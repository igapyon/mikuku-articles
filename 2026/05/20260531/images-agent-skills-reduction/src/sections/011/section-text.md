# 011 中品質の大量データは効くけれど重い

## 主題

中品質の大量データは Agent を助けるが、毎回全部読むと重い。

## 重要キーワード

- コンテンツ型 Agent Skills
- 背景知識
- 判断基準
- テンプレート
- 過去例
- 作業ログ
- 判断メモ
- 情報を置く
- 情報へアクセスする

## 価値

- 情報が多少粗くても材料になる
- AI agent が拾って整理できる
- 過去の知見を再利用できる

## 問題

```text
大量の資料
  ↓
毎回全部読む
  ↓
トークン消費が増える
```

## 解決の方向

情報を置くことと、アクセスすることを分ける。

## 吹き出し案

「資料は嬉しいけれど、全部読むと重くなるのです…」
