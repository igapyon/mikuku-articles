# 010 index と front matter で読む候補を絞る

## 主題

資料が増えたら、本文を全部読む前に候補を絞る入口を作る。

## 重要キーワード

- `index.json`
- front matter
- ファイル名
- 見出し
- `category`
- `description`
- miku-indexgen
- 軽い地図

## 候補選びの流れ

```text
index / front matter / ファイル名
  ↓
候補を選ぶ
  ↓
必要な本文だけ読む
```

## 注意

- front matter も長すぎると重い
- ファイル名は短すぎても長すぎても扱いづらい
- 軽く、でも効く程度に構造化する

## 吹き出し案

「まず地図を見てから、本文へ進むのです…」
