# 013 examples と templates を分ける

## 主題

`templates/` は構造、`examples/` は温度感や粒度を伝える。

## 重要キーワード

- `references/`
- `templates/`
- `examples/`
- 構造
- 温度感
- 粒度
- 期待品質
- 避けたい例

## 役割分担

| 要素 | 役割 |
| --- | --- |
| `references/` | 判断基準・背景知識・文体ルール |
| `templates/` | 出力構造を安定 |
| `examples/` | 温度感・粒度・境界を伝える |

## 対比

```text
templates = 形
examples  = 温度
```

## 吹き出し案

「形と温度感は、分けて置くと見えやすいです…」
