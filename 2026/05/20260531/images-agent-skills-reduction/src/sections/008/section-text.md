# 008 必要になった段階で読む

## 主題

必要になるまで奥の情報は読ませず、作業段階に応じて参照を開く。

## 重要キーワード

- progressive disclosure
- 必要な段階
- 必要な情報だけ
- `SKILL.md`
- `article-writing.md`
- `graphic-recording.md`
- `assets/`
- 距離感

## 段階的に読む

```text
通常作業 → SKILL.md
記事執筆 → references/article-writing.md
グラレコ → references/graphic-recording.md
画像や素材 → assets/
```

## 合言葉

情報は残す。でも、毎回全部は読ませない。

## 吹き出し案

「必要になってから、次の扉を開くのです…」
