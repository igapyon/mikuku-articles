# 009 references をただの置き場にしない

## 主題

`references/` は詳細を逃がす場所だが、毎回全部読むなら重さは変わらない。

## 重要キーワード

- `references/`
- 用途別
- 読む候補
- 棚のラベル
- `archive/`
- `legacy/`
- 通常探索範囲

## 用途別の分け方

- 記事執筆用
- レビュー用
- グラレコ用
- 運用ルール用
- 過去ログ用
- 蒸留済み資料用

## 図解

```text
SKILL.md
  ↓ 用途に合う参照を選ぶ
references/
  ├ article
  ├ review
  ├ graphic-recording
  ├ distilled
  └ archive
```

## 吹き出し案

「置き場所を変えるだけでは、まだ半分なのです…」
