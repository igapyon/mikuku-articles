# Image Generation Prompt

Use case: infographic-diagram
Asset type: cover-like wide graphic-recording poster for a Japanese technical Note article
Primary request: Create a single wide horizontal cover graphic / graphic recording image explaining the article "生成AI agent の向こう側には、いろいろな妖精さんがいる".

Most important cover requirement:

The cover must visually read as "cute rabbit helpers behind a generative AI agent." In this article series, the fairy helpers are rabbit-shaped. The first impression of the cover should be four cute rabbit helpers, not abstract diagrams, not generic fairies, and not Mikuku alone.

Make the four rabbit helpers large and central enough to define the cover:

- CLI の妖精さん: bright light-brown Netherland Dwarf rabbit.
- API の妖精さん: white and pale-gray Holland Lop rabbit.
- データベースの妖精さん: near-black chocolate Mini Rex rabbit.
- MCP server 内部処理の妖精さん: cream-colored Jersey Wooly rabbit.

Place these four rabbits as the main visual cluster in the left-center or center of the poster. Add small technical labels and icons around each rabbit. The rabbits are the cover hook.

Canonical character prompt for Mikuku:

masterpiece, best quality, ultra-detailed anime portrait, head-and-shoulders 1girl, shy uneasy expression, slight 3/4 view, looking to her right and slightly upward.

Warm chestnut-brown eyes with gold flecks, soft blush, worried upward-slanted brows, small downturned pout, fair warm skin, delicate V-U shaped chin.

Soft caramel-chestnut very long twin-tails tied high with black ribbons and deep-red trim, silky wavy hair, asymmetrical curtain bangs, long side strands framing cheeks.

Thin clean line art, soft cel shading, watercolor-like shading, subtle rim light, warm minimal background.

Character identity requirements:

- Same character: Mikuku / みくく.
- Do not redesign Mikuku.
- Preserve character identity, canonical character reference, face shape, eyes, caramel-chestnut twin-tails, black ribbons with deep-red trim, shy gentle expression, and soft anime tech-explainer mood.
- Mikuku should be on the right side of the poster, looking toward the diagrams on the left and center.
- Do not let Mikuku hold any objects.
- Hands should be simple, natural, and not emphasized.

Poster concept:

A warm, hand-drawn Japanese graphic recording poster. Mikuku gently explains that beyond a generative AI agent there are many outside helpers that actually do work. The poster should show the difference between:

- Agent Skills = 魔法書
- MCP = 魔法陣
- CLI / API / Database / internal function = 妖精さん

Layout:

- Wide horizontal poster, 16:9.
- Background: clean warm off-white paper, light notebook texture, soft pastel accents.
- Center: four cute rabbit-shaped fairy helpers as the strongest cover visual.
- Left and lower areas: compact graphic recording map and role labels.
- Right: Mikuku, smaller than the rabbit cluster, head-and-shoulders to half-body anime-style presenter, not holding anything.
- Use hand-drawn boxes, arrows, small icons, speech bubbles, and clean visual grouping.

Main diagram:

Put a large flow across the center:

ユーザー → 生成AI agent → MCP 魔法陣 → 妖精さんたち → 結果

Under each stage, add short Japanese labels:

- 生成AI agent: 考える / お願いを作る
- MCP: 外へ届ける通り道
- 妖精さん: 実際に仕事をする
- 結果: 記録 / ファイル / 回答

Rabbit-fairy categories section:

Create four grouped panels around the central rabbit cluster. The fairy helpers in this article series must be rabbit-shaped. Do not draw generic fairies, abstract mascots, sparkles-only helpers, or human-like fairies. Each helper is a rabbit matched to the article table.

1. CLI の妖精さん
   - Draw as a bright light-brown Netherland Dwarf rabbit.
   - Compact, upright, round-faced rabbit with short ears.
   - Small terminal icon nearby.
   - コマンド
   - 引数
   - 作業ディレクトリ
   - 入力と出力

2. API の妖精さん
   - Draw as a white and pale-gray Holland Lop rabbit.
   - Soft drooping ears, gentle expression.
   - Small network/cloud letter icon nearby.
   - URL
   - method
   - parameter
   - 認証

3. DB の妖精さん
   - Draw as a near-black chocolate Mini Rex rabbit.
   - Sleek plush coat, calm and precise expression.
   - Small database cylinder icon nearby.
   - SQL / query
   - 条件
   - 権限
   - 記録を探す

4. MCP 内部処理の妖精さん
   - Draw as a cream-colored Jersey Wooly rabbit.
   - Fluffy woolly coat, small careful-worker feeling.
   - Small gear/checklist icon nearby.
   - 入力検査
   - 文字列整形
   - 軽い計算
   - ローカル情報

Rabbit identity requirements:

- The helper fairies are rabbits in this series.
- Keep the four rabbit types visually distinct.
- Use soft, cute, article-friendly rabbit illustrations.
- The rabbits are the main cover motif; Mikuku remains the presenter on the right.
- Do not replace the rabbits with winged fairies, humanoids, or vague abstract creatures.

Comparison section:

Make a compact comparison table or three connected cards:

- 魔法書 = Agent Skills = 前提・文体・手順
- 魔法陣 = MCP = お願いの通り道
- 妖精さん = 外部処理 = 実行する相手

Boundary and troubleshooting section:

Show a lower band titled:

万能ではなく、境界がある

Include small check bubbles:

- お願いが届いた？
- 材料は足りた？
- 権限はある？
- tool はある？
- 結果を読み違えていない？

Mikuku speech bubble:

Add one readable speech bubble near Mikuku:

「あ、あの…agent が全部ひとりでしているわけではないのです…」

Style:

- Japanese anime-style educational infographic.
- Graphic recording illustration, hand-drawn notebook, soft cel shading, warm light.
- Information-rich but not cluttered.
- Use readable short Japanese labels where possible.
- Soft brown, gentle teal, muted red, light gold, and gray-blue accents. Avoid a one-note palette.
- No corporate slide look, no photorealism, no dark sci-fi mood, no hard UI dashboard.
- No watermark, no logo, no tiny unreadable body paragraphs.

Text accuracy guidance:

Text may be simplified, but the main Japanese labels should be legible and conceptually correct. Prefer short labels over long sentences.
