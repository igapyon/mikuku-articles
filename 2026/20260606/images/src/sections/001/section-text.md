# 001: はじめに

## 主題

生成AI agent の向こう側で働く「妖精さん」を、うさぎ型の見立てで紹介する入口。

## 図解キーワード

- 生成AI agent
- MCP = 魔法陣
- 妖精さん = 外側で仕事をする相手
- うさぎ型の見立て
- CLI / API / DB / MCP 内部処理

## 重要な表

| 妖精さん | うさぎ型の見立て |
| --- | --- |
| CLI | 明るい茶色のネザーランドドワーフ |
| API | 白と淡いグレーのホーランドロップ |
| データベース | 黒に近いチョコレート色のミニレッキス |
| MCP server 内部処理 | クリーム色のジャージーウーリー |

## 構図案

中央に4羽のうさぎ型妖精さん。
背後に MCP の魔法陣。
右側にみくくが案内役として説明する。
