# 005: データベースの奥で記録を探す妖精さん

## 主題

データベースの妖精さんは、記録庫の奥から条件に合う情報を探してくる。

## うさぎ型

- 黒に近いチョコレート色のミニレッキス
- 静かで正確
- 記録の棚を落ち着いて探す雰囲気

## 図解キーワード

- SQL
- NoSQL query
- query DSL
- 専用 API
- 接続先
- 認証情報
- 権限
- 条件
- 並び順

## 流れ

```text
条件
  ↓
SQL / query
  ↓
記録を探す
  ↓
必要な項目だけ返す
```

## 注意点

- 何でも見られるわけではない
- 許された範囲で探す
- 想像で補わず、記録へ聞きに行く

## 吹き出し

「記録がある場所へ、そっと確かめに行きます…」
