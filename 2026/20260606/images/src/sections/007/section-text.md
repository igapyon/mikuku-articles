# 007: 妖精さんは、自分で働くことも、取り次ぐこともある

## 主題

妖精さんは、自分で処理を完結する場合も、別の仕組みへお願いを取り次ぐ場合もある。

## 図解キーワード

- 自分で処理する
- 別の処理へ取り次ぐ
- CLI を呼ぶ
- API へ問い合わせる
- DB へ query を投げる
- 入れ子構造
- 境界

## 入れ子図

```text
agent
  ↓
うさぎ型の妖精さん
  ├─ 自分で処理
  └─ 別のうさぎ / 外部処理へ取り次ぎ
```

## 注意点

- 細かく追いすぎると迷子になる
- まず agent から見てお願いできる相手を妖精さんとして見る
- 境界を分けると落ち着いて読める

## 吹き出し

「自分で働く子も、取り次ぐ子もいるのです…」
