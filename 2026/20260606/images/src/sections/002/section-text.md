# 002: 妖精さんは、実際に仕事をしてくれる相手

## 主題

妖精さんは agent そのものでも MCP そのものでもなく、実際に外側の仕事をする相手。

## 図解キーワード

- ファイルを読む
- 検索する
- データベースを調べる
- API へ問い合わせる
- 計算する
- 変換処理を実行する

## 関係図

```text
agent が考える
  ↓
MCP がお願いを届ける
  ↓
うさぎ型の妖精さんが実行する
  ↓
結果が返る
```

## 誤解しやすい点

- 生成AIが全部ひとりでしているわけではない
- 会話の中で考える部分と、外側で実行する部分が分かれている
- 妖精さんは「実行担当」

## 視覚モチーフ

うさぎ型妖精さんが、ファイル、虫眼鏡、DB、API封筒、変換矢印の周りで働く。

## 吹き出し

「会話の外側で、そっと仕事をしてくれる相手なのです…」
