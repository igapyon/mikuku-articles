# 003: CLI の姿をした妖精さん

## 主題

CLI の妖精さんは、コマンドラインから呼び出す道具を実行する。

## うさぎ型

- 明るい茶色のネザーランドドワーフ
- 小さく丸い姿
- 端末の近くでぱたぱた働く

## 図解キーワード

- コマンド名
- 引数
- 作業ディレクトリ
- 入力ファイル
- 出力先
- git / ffmpeg / mvn / npm

## 流れ

```text
お願い
  ↓
コマンド + 引数
  ↓
CLI 実行
  ↓
標準出力 / ファイル出力 / JSON
```

## 注意点

- ふわっと何でも察するわけではない
- 材料がそろうと安定して働く
- agent から見るとお願いしやすい道具

## 吹き出し

「コマンド名と材料がそろうと、ちゃんと働けます…！」
