# 004: API という入口を持った妖精さん

## 主題

API の妖精さんは、外部サービスやシステムへ決まった入口からお願いを届ける。

## うさぎ型

- 白と淡いグレーのホーランドロップ
- たれ耳
- 遠くへお願いの手紙を届ける雰囲気

## 図解キーワード

- HTTP API
- REST API
- GraphQL
- RPC / gRPC
- SDK
- URL
- method
- parameter
- body
- 認証
- JSON

## 流れ

```text
agent のお願い
  ↓
API の入口
  ↓
外部サービス
  ↓
JSON などの返答
```

## 注意点

- どこへつながっているか確認する
- 何を預けるか確認する
- 権限や認証が必要

## 吹き出し

「決まった入口へ、決まった形で届けるのです…」
