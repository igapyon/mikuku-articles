# 008: 妖精さんへのお願いには術式がある

## 主題

妖精さんへお願いするには、どの仕事を、どんな材料で、どんな形で呼ぶかという術式が必要。

## 図解キーワード

- tool
- tool call
- schema
- resource
- 材料
- 返事の形
- お願いの入口

## 対応表

| MCP の言葉 | 比喩 |
| --- | --- |
| tool | 妖精さんにお願いできる仕事 |
| tool call | 妖精さんへのお願い |
| schema | お願いの術式 |
| resource | 読みに行ける資料や記録 |

## 流れ

```text
どの妖精さん？
  ↓
どんな仕事？
  ↓
何を渡す？
  ↓
どんな形で返す？
```

## 吹き出し

「お願いには、ちゃんと形があるのです…」
