# 006: MCP server の中に住む小さな処理の妖精さん

## 主題

妖精さんは外部 CLI や API だけではなく、MCP server 内部で完結する小さな処理でもよい。

## うさぎ型

- クリーム色のジャージーウーリー
- ふわふわ
- 小さな机でこつこつ整える雰囲気

## 図解キーワード

- 入力検査
- 文字列整形
- 軽い計算
- ローカル情報の返却
- 決まった形式への変換

## 対比

| 外へ取り次ぐ | その場で整える |
| --- | --- |
| CLI / API / DB | MCP server 内部処理 |

## メッセージ

小さな処理も、agent から見れば立派な実行担当。

## 吹き出し

「近くにいる子も、ちゃんと働いているのです…」
