# 011: おわりに

## 主題

生成AI agent の外部連携を、魔法書、魔法陣、うさぎ型の妖精さんとして見ると、役割分担の輪郭が見える。

## 図解キーワード

- ひとりで全部しているわけではない
- 外側へお願いを届ける
- 決められた形で呼ぶ
- 決められた範囲で働く
- 術式、材料、権限、結果の扱い
- 見分ける力

## まとめ図

```text
どの妖精さんに
何を
どんな形で
お願いしているか
```

## メッセージ

ただ便利と見るだけではなく、お願いの相手と形を見分けることが大事。

## 吹き出し

「少しずつ見分けられることが、大事な足場なのです…」
